# 无尽深渊（Abyssal Dive）整合策划案（可迭代版）

> 文档用途：统一项目目标、当前实现、后续扩展边界，供开发与答辩口径共用。  
> 最新更新：2026-03-12

---

## 0. 版本基线

- 项目定位：Unity 2D 纵向下坠生存跑酷 + 轻 Roguelike
- 开发基线：左右移动 + 下坠触敌自动反冲（无主动攻击键）
- 演示目标：可运行、可演示、可解释

---

## 1. 核心体验与流程

### 1.1 叙事背景（不进CG）
莱姆在探寻古老深渊遗迹时触发重力机关，坠入无尽深渊。深渊的起源、机关真相与能力来源隐藏于沿途收集品与生物图鉴中，等待逐步解锁。

### 1.2 首次游玩流程（教程+CG）
- CG1（开局）：遗迹入口探查 -> 地面崩塌 -> 坠入黑暗，提示“深渊机关已触发，唯有下坠求生”
- 教程一：高空碎台（遗迹上层残骸），低速引导基础操作
- CG2（休息平台）：深渊能量融入，提示“获得特殊能力”
- 教程二：晶光通道（遗迹中层），引导能力使用
- 教程结束对话：返回主界面 / 开启无尽模式
- CG3（开启无尽模式）：穿过晶光通道进入黑暗深渊，提示“新手教程已完成，无尽挑战开启”

### 1.3 正式游戏（无尽模式）
- 单一背景：黑暗深渊
- 下坠速度持续递增且无上限
- 收集品与图鉴碎片解锁深渊背景与能力来历
- 目标：尽可能下坠更深处

### 1.4 核心循环（正式模式）
主菜单 -> 开始 -> 下坠生存 -> 踩踏特殊生物获得能力 -> 结算 -> 重开/返回主菜单

## 2. 功能模块口径

### 2.1 玩家与碰撞
- 重力递增与下落上限控制
- 左右移动与边界阻挡
- 触敌反冲
- 障碍扣血 + 减速 + 障碍消失
- 高速防穿透

### 2.2 特殊生物与能力
- 特殊生物生成：每 100 分生成一只，从生物池随机选择
- 踩踏击杀直接获得对应能力，并替换当前能力
- 踩踏击杀触发暂停与能力描述栏，点击任意位置恢复
- 踩踏移动反馈与踏板砖块一致
- 主动能力：屏幕正下方按钮可点击释放
- 被动能力：按钮置灰不可点击

### 2.3 图鉴（Codex）
- 图鉴四类页面：收藏品 / 障碍 / 跳板 / 生物
- 图鉴内四按钮切换，默认显示“生物”图鉴
- 未遇到的生物显示“?”占位且不可点击
- 解锁逻辑以“获得/遭遇记录”为主

### 2.4 菜单与HUD
- 主菜单四页面：Leaderboard / Codex / Achievements / Settings
- HUD 显示：分数、血量、速度（SPD）、加速度（ACC）
- HUD 增加能力按键（屏幕正下方）
- 暂停与结算菜单路径完整

### 2.5 数据与存档
- `run_progress.json` 存储分数、局数、模式、图鉴解锁与排行榜
- 模式解锁：Classic（默认）/ Hard（400分+3局）/ Abyss（900分+8局）

### 2.6 表现层
- 砖块 Sprite、动画与显示特效补齐
- 关键音效闭环：受击 / 能力获取 / 结算
- UI 文案与视觉规范统一

## 3. 数值与回归口径

- 特殊生物生成间隔：每 100 分
- 加速度分档（随分数/深度）：<500 为 0.005；500-1999 为 0.01；>=2000 为 0.05
- 图鉴解锁进度以“获得/遭遇记录”为准（四类条目）
- 回归标准：
  - 连续 20 次“开始 -> 死亡 -> 重开”无阻断
  - 10 局中 >= 8 局在 3-5 分钟区间

## 6. 风险与保底策略

- 风险：内容扩展引入新 Bug
- 对策：新增内容先在测试场景回归，再并入主线
- 风险：难度曲线不稳定
- 对策：10 局模板 + 小步快调，禁止一次性大改
- 风险：演示版不稳定
- 对策：设封板版本，扩展项不达标时不影响主版本
